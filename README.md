# CAN Attack Detect

This project simulates CAN bus attacks and detection/mitigation strategies in a Linux SocketCAN environment.

## Planned Features
- Legitimate CAN sender node
- Receiver/dashboard node
- Spoofing attack node
- Detection logic
- Logging and analysis
- GUI for real-time signal display

## Environment
- Ubuntu VM
- SocketCAN
- Virtual CAN (vcan0)
- Python
